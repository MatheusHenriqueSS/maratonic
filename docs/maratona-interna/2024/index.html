A. Cubo mágico da Showpee​

Primeiramente, note que é possível arranjar os elementos da matriz como você quiser: coloque o elemento que você quiser na posição (0,0)(0, 0)(0,0), depois outro na posição (0,1)(0, 1)(0,1), depois na posição (0,2)(0, 2)(0,2), e assim por diante até a posição (n,m)(n, m)(n,m).

Já que posso arranjar os elementos como quiser, basta analisar se o conjunto dos números da primeira matriz é igual ao conjunto da segunda matriz. Para fazer isso, coloque todos os elementos da primeira matriz num array e ordene-o. Faça a mesma coisa para a segunda matriz e compare se os dois arrays são iguais.
